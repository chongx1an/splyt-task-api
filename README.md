## Introduction
This Node API project is used to serve the request from this [Client project](https://github.com/chongx1an/splyt-task-client)

## Setup instruction

1. Installation:\
`npm install`

2. Start the project:\
`npm run start`

3. Project are default running at port 4000\
http://localhost:4000/

## Perform unit tests
`npm run test`
